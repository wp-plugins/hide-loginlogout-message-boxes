== Description ==

This plugin will hide the yellow (default) message box(es) on the login & logout screens. 

Find out more at: http://www.mretzlaff.com/wordpress-plugins/wp-plugin-hide-loginlogout-message-boxes/

== Installation ==

1. Unzip the downloaded ZIP archive file to a folder
2. Upload the folder "backend-logo" to your WordPress Plugin folder "/wp-content/plugins/"
3. Activate the plugin through the "Plugins" menu in WordPress (Backend)
